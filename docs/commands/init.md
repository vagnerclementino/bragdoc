# Init

### init

Initialize Bragdoc by creating the configuration directory and setting up the database with migrations

```bash
Usage:
  bragdoc init [flags]

Flags:
  -c, --company string     Your company (optional)
  -e, --email string       Your email (required)
  -h, --help               help for init
  -j, --job-title string   Your job title (optional)
  -l, --locale string      Locale (language-COUNTRY): en-US or pt-BR (default "en-US")
  -n, --name string        Your full name (required)
```
